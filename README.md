# 🌙 Sobre o Projeto

Este projeto recria o universo de Enrolados em formato de site interativo, combinando estética cinematográfica com elementos visuais e dinâmicos.

A proposta é transformar o navegador em uma experiência imersiva dentro do Reino de Corona, explorando luz, fantasia e narrativa visual.

---

# ✨ O que o site apresenta

- 👑 Personagens principais do universo  
- 🏰 Reino de Corona com ambientação visual  
- 🏮 Festival das Lanternas com efeito animado  
- 📖 Livro mágico interativo em formato de capítulos  
- 🎶 Trilha sonora temática  
- ✉️ Mensagens enviadas em lanternas  
- 🌌 Cenário com estrelas, partículas e lanternas flutuantes  
- 🌗 Alternância entre modo dia e noite

# 🏰 Seções do Site

## 👑 Hero
Introdução ao universo com estética cinematográfica.

## 📖 Livro Mágico
Seção em formato de livro com navegação por capítulos.

## 🧝 Personagens
Rapunzel, Flynn, Pascal, Maximus e Gothel.

## 🏰 Reino de Corona
Mapa estilizado com elementos do universo.

## 🎶 Trilha Sonora
Player com músicas que reforçam a ambientação.

## 🏮 Lanternas
Efeito visual interativo com lanternas flutuantes e mensagens.

Elementos visuais que representam pontos da história.
