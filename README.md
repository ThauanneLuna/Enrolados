# 👑 Enrolados — Um Reino de Luz

> ✨ Uma experiência interativa inspirada no universo de *Enrolados (Tangled)*, recriando a magia de Corona, suas lanternas e seus personagens em um site imersivo.

---

## 📌 Sobre o Projeto

Este site é uma homenagem ao filme *Enrolados*, desenvolvido como uma experiência visual e interativa focada em atmosfera, narrativa e estética.

Ele não é apenas informativo — é uma imersão no reino de Corona.

### ✨ O que o site oferece

- 🌙 **Abertura cinematográfica** com lanternas e animação inicial
- 🌓 **Modo dia/noite** com mudança completa de atmosfera
- 🖱️ **Cursor personalizado** com efeitos de brilho e partículas
- 📖 **Livro mágico de navegação** com capítulos do reino
- 👑 **Seção de personagens** com cards visuais e quotes
- 🗺️ **Mapa do Reino de Corona** com locais icônicos
- 🎶 **Trilha sonora interativa** do filme
- 🧠 **Simbolismo e análises** sobre a história
- 🏮 **Mensagens nas lanternas** (interação do usuário)
- ✨ **Easter eggs escondidos** pelo site

---

## 🚀 Tecnologias Utilizadas

- HTML5 — estrutura do site
- CSS3 — animações, efeitos visuais e responsividade
- JavaScript — interatividade e efeitos dinâmicos

---

## 🎯 Objetivo

Criar uma experiência web que mistura design, narrativa e emoção, trazendo a sensação do filme para dentro da tela.

---

## ⚠️ Aviso

Este é um projeto de fã, sem fins comerciais.  
Todos os direitos do universo *Enrolados* pertencem à Disney.
